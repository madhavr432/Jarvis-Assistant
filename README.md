# Jarvis-Assistant
Jarvis Assistant is a voice‑controlled AI assistant built in Python. It integrates speech recognition, text‑to‑speech, and automation libraries to provide a hands‑free experience for everyday tasks.
Features
Voice Commands: Recognizes English and Hindi input using Google Speech Recognition.

Text‑to‑Speech: Responds with natural voice output using pyttsx3.

Automation: Executes tasks like opening applications, searching the web, sending WhatsApp messages, and controlling system functions.

Multilingual Support: Handles both English and Hindi commands for broader usability.

Extensible Design: Modular code structure allows easy addition of new commands and features.

🛠 Tech Stack
Python (Core language)

speech_recognition (Voice input)

sounddevice (Audio recording, replacing PyAudio)

pyttsx3 (Text‑to‑speech)

pyautogui (Automation)

pywhatkit (WhatsApp & YouTube integration)

googletrans (Translation support)
